# Nectar

Nectar's Laravel Backend Server

## Installation

Use the dependency manager [composer](https://getcomposer.org/download/) to install Nectar.

In the root directory of the project run the commands below:

```bash
composer install
```

Migration database

```bash
php artisan migrate
```

Run Seeder
```bash
php artisan db:seed
```

Start server in localhost

```bash
php artisan serve
```
